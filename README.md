# statscsv

AP Stats project — performs linear regression between two CSV datasets and outputs a scatter plot with regression line + statistical variables.

## CSV Format

All input CSVs must follow this structure:

```
Entity,Code,Year,Value
United States,USA,2020,12345.6
...
```

## Usage

```
python main.py
```

You will be prompted to:

1. Enter a target year
2. Select the explanatory variable CSV (file dialog)
3. Select the response variable CSV (file dialog)
4. Enter a graph title and axis labels

## Output

- Regression graph saved to `statscsv/output/graph.png`
- Stats printed to console: slope, intercept, r, p-value, std error, r²

## Dependencies

```
pip install numpy scipy matplotlib
```

> tkinter is included with most Python installations.

## Notes

- Countries are matched by their 3-letter code (the `Code` column)
- Countries missing data in either dataset are excluded
- Non-numeric values are skipped with a warning
